# Аналитика рынка недвижимости (SQL-проект)

## Описание
Данный проект содержит комплексный SQL-анализ базы данных агентства недвижимости. Проект разделен на 15 модулей, каждый из которых проверяет отдельную бизнес-гипотезу.

## Стек технологий
* **СУБД:** PostgreSQL / MySQL
* **Инструменты:** SQL Queries, CTE (Common Table Expressions), Subqueries.

## Основные этапы анализа
1. **Анализ персонала:** Оценка эффективности и расчет комиссионных.
2. **Рыночный анализ:** Поиск инвестиционно-привлекательных и переоцененных объектов.
3. **Клиентская аналитика:** Идентификация паттернов поведения покупателей.
4. **Региональная динамика:** Сравнение районов по стоимости квадратного метра.

## Структура репозитория
Все запросы находятся в папке `/queries`. Каждый файл содержит пояснение бизнес-ценности гипотезы.
